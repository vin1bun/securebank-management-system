# securebank-management-system
A secure banking application developed in Python using OOP architecture, encrypted PIN authentication (SHA-256), persistent JSON storage, and transaction history management.
## Features
- Account creation with validation
- SHA-256 hashed PIN security
- Deposit & withdrawal operations
- Transaction history tracking
- JSON-based persistent storage

## Tech Stack
- Python
- JSON
- hashlib
- OOP

##  How to Run

1. Clone the repository
2. Navigate to project folder
3. Run:

python main.py
